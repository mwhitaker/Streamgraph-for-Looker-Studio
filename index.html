Stream Graph Visualization - Development Preview

Sample Data Structure:

This visualization expects CSV data with three columns:

  • Date column: Contains 'date', 'time', 'year', 'month', or 'day' in the name
  • Category column: Contains 'category', 'sex', 'gender', 'group', 'type', or 'name' in the name
  • Value column: Contains 'value', 'count', 'amount', 'prop', 'rate', or 'percent' in the name

Currently configured to load: data/babynames.csv (primary) or data/unemployment.csv (fallback)

To add your own CSV file:

  1. Place your CSV file in the data/ directory
  2. Edit src/config.js and add your file path to sampleDataPaths array
  3. Restart the development server with npm run dev
